# Multiple Linear Regression - Exam Score Prediction

This project predicts students' **exam scores** using a **Multiple Linear Regression** model based on study and lifestyle-related features.  
It includes both a model development notebook (`.ipynb`) and a user-friendly **Streamlit interface**.

---

## Project Objective

Predict `Exam Score` using the following input features:

- Study Hours
- Sleep Hours
- Attendance Rate
- Social Media Hours

This enables quick analysis of how different student habits may affect exam performance.

---

## Project Structure

- `2-MultipleLeinearRegression.ipynb`  
  Notebook for model development and exploratory analysis.

- `2-multiplegradesdataset.csv`  
  Dataset used for training and evaluation.

- `app.py`  
  Streamlit-based user interface (prediction + visual analytics).

- `requirements.txt`  
  Python dependencies required for the project.

---

## Tech Stack

- Python
- Pandas
- Scikit-learn
- Streamlit
- Plotly
